# Redundant-Campus-Area-Network
This project demonstrates a redundant enterprise network based on the Cisco three-tier architecture (Core, Distribution, Access) The lab includes VLAN segmentation, inter-VLAN routing, OSPF, HSRP, EtherChannel, Rapid PVST+, ACLs, DHCP, DNS, NTP, SNMP, Syslog, FTP, SSH, NAT, and wireless connectivity.

# Project Summary
This project simulates a redundant enterprise campus network built using the Cisco three-tier architecture (Core, Distribution, and Access layers). It demonstrates key enterprise networking concepts such as VLAN segmentation, inter-VLAN routing, and dynamic routing with OSPF. High availability is achieved through HSRP, EtherChannel, and Rapid PVST+ to ensure network resilience during link or device failures. The lab also integrates essential network services including DHCP, DNS, NTP, SNMP, Syslog, NAT, and secure management protocols like SSH. The goal of the project is to model a scalable, secure, and fault-tolerant enterprise network environment.
