# Redundant-Campus-Area-Network
This project demonstrates a redundant enterprise network based on the cisco three-tier architecture (Core, Distribution, Access) The lab includes VLAN segmentation, inter-VLAN routing, OSP, HSRP, EtherChannel, Rapid PVDT+, ACLS, DHCP, DNS, NTP, SNMP, Syslog, FTP, SSH, NAT, and wireless connectivity.
